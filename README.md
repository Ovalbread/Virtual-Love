# Virtual-Love

An interactive React birthday website with:
- Username/password access gate
- Cute click interactions (capybara hug, duck waddle, butterfly swarm, tulip burst)
- Final celebration with poppers
- Virtual scrapbook with animated page turns

## Run

```bash
npm install
npm run dev
```

## Login Credentials

Edit in `src/App.jsx`:
- `AUTH.username`
- `AUTH.password`
